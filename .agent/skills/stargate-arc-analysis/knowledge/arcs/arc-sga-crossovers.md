# Arc : CROSSOVERS SG-1 / ATLANTIS 

**Séries :** SG-1 + SGA · **Code :** `CROSS`  
**Portée complète :** SG-1 S8 → SGA S4 (2004–2008)  
**Couleur :**  Alliance inter-séries

> Arc des points de convergence entre les deux programmes. Les crossovers ne sont pas anecdotiques — ils marquent des transitions narratives majeures : bifurcation initiale, interdépendance des menaces, puis unification du commandement.
